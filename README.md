# Weather Diary (Дневник погоды)

## Автор
Масленников Кирилл Владимирович

## Описание программы
Weather Diary - это GUI-приложение для ведения дневника погоды. Программа позволяет пользователям записывать наблюдения за погодой, просматривать их в удобной таблице, фильтровать по дате и температуре.

### Основные функции:
- Добавление записей с указанием даты, температуры, описания и осадков
- Просмотр всех записей в табличном виде
- Фильтрация записей по дате
- Фильтрация записей по температуре (показать выше указанной)
- Сохранение данных в JSON формате
- Автоматическая загрузка данных при запуске

## Как использовать

### Запуск программы
1. Убедитесь, что у вас установлен Python 3.6 или выше
2. Скачайте файлы программы
3. Запустите файл `weather_diary.py`:
```bash
python weather_diary.py
Добавление записи
Введите дату в формате ДД.ММ.ГГГГ (по умолчанию текущая дата)

Введите температуру в градусах Цельсия

Введите описание погоды (например, "солнечно", "дождь")

Выберите наличие осадков (да/нет)

Нажмите кнопку "Добавить запись"

Фильтрация записей
По дате: введите дату в поле "Дата" в секции "Фильтрация"

По температуре: введите минимальную температуру в поле "Температура от"

Нажмите "Применить фильтр" для отображения отфильтрованных данных

Используйте кнопку "Сбросить" для отображения всех записей

Примеры использования
Пример 1: Базовое использование
Добавьте запись: 15.01.2024, -5°C, "Снежно", осадки: да

Добавьте запись: 16.01.2024, 0°C, "Пасмурно", осадки: нет

Добавьте запись: 17.01.2024, +5°C, "Солнечно", осадки: нет

Программа отобразит все 3 записи

Пример 2: Фильтрация по дате
Введите дату "16.01.2024" в фильтре

Нажмите "Применить фильтр"

Отобразится только запись за 16 января

Пример 3: Фильтрация по температуре
Введите "Температура от: 0"

Нажмите "Применить фильтр"

Отобразятся записи с температурой 0°C и выше

Тестирование валидации
Попробуйте ввести неверную дату (например, 31.13.2024) - будет показана ошибка

Попробуйте ввести текст вместо температуры - будет показана ошибка

Попробуйте оставить поля пустыми - будет показана ошибка

Техническая информация
Данные сохраняются в файл weather_data.json в формате JSON

Приложение использует стандартную библиотеку tkinter для GUI

При первом запуске создается пустой список записей
