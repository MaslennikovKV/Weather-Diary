# Weather Diary (Дневник погоды)

**Автор:** Масленников Кирилл Владимирович 

## Описание
GUI-приложение для ведения дневника погоды. Позволяет добавлять записи с датой, температурой, описанием и признаком осадков, просматривать их в таблице, фильтровать по дате и температуре, а также сохранять данные в JSON-файл и загружать из него.

## Требования
- Python 3.6 или выше
- Стандартная библиотека Tkinter (обычно входит в установку Python)

## Как запустить
1. Склонируйте репозиторий или скачайте файлы.
2. Убедитесь, что Python установлен.
3. Запустите скрипт:
   ```bash
   python main.py
Приложение откроется в отдельном окне.

Использование
Добавление записи: заполните поля Дата (ГГГГ-ММ-ДД), Температура (число), Описание (не пустое), отметьте чекбокс "Осадки" при необходимости и нажмите "Добавить запись".

Просмотр: все записи отображаются в таблице.

Фильтрация:

По дате: введите дату в поле "Дата" в блоке фильтров и нажмите "Применить".

По температуре: введите минимальную температуру и нажмите "Применить". Показываются записи с температурой ≥ указанной.

Кнопка "Сбросить" отменяет все фильтры.

Сохранение/загрузка: кнопки "Сохранить в JSON" и "Загрузить из JSON". Файл weather_data.json создаётся рядом со скриптом.

Очистка таблицы: удаляет все строки из вида (но не из памяти, пока не загружены новые данные).

Примеры использования (тесты)
Добавьте запись: 2026-05-01, 15, "Облачно", осадки нет.

Добавьте запись: 2026-05-02, 8, "Дождь", осадки да.

Примените фильтр по дате: 2026-05-01 – отобразится только первая запись.

Примените фильтр по температуре ≥ 10 – отобразится только запись с температурой 15.

Сохраните данные, закройте приложение, откройте снова – данные загрузятся автоматически.
