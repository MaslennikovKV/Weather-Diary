Масленников Кирилл Владимирович

## Описание программы
Weather Diary — это GUI-приложение для ведения дневника погоды. Программа позволяет пользователям записывать ежедневные наблюдения за погодой, включая температуру, описание погодных условий и наличие осадков. Все записи сохраняются в JSON-формате и могут быть отфильтрованы по различным критериям.

### Основные функции:
- Добавление записей о погоде с указанием даты, температуры, описания и наличия осадков
- Просмотр всех записей в удобной таблице
- Фильтрация записей по дате (выбор конкретного дня)
- Фильтрация по температуре (например, показать записи выше +10°C)
- Фильтрация по наличию осадков
- Редактирование и удаление существующих записей
- Автоматический подсчет статистики (средняя, максимальная, минимальная температура)
- Сохранение данных в JSON формате
- Автоматическая загрузка данных при запуске
- Валидация вводимых данных

## Как запустить приложение

### Требования
- Python 3.6 или выше
- Стандартные библиотеки Python (tkinter, json, datetime, pathlib)

### Инструкция по запуску
1. Убедитесь, что у вас установлен Python 3.6 или выше:
```bash
python --version
Скачайте или клонируйте репозиторий:

bash
git clone https://github.com/username/weather-diary.git
cd weather-diary
Запустите приложение:

bash
python weather_diary.py
При первом запуске будет создан файл weather_data.json для хранения записей.

Как использовать
Добавление новой записи
Введите дату в формате ДД.ММ.ГГГГ (по умолчанию установлена текущая дата)

Введите температуру в градусах Цельсия (например: 23.5 или -5)

Введите описание погоды (например: "солнечно", "облачно", "дождь")

Выберите наличие осадков (Да/Нет)

Нажмите кнопку "Добавить запись"

Фильтрация записей
По дате: введите конкретную дату в поле "Дата" в секции фильтрации

По температуре: введите минимальную температуру в поле "Температура от"

По осадкам: выберите "да", "нет" или "все" в выпадающем списке

Нажмите "Применить фильтр" для отображения отфильтрованных данных

Используйте "Сбросить фильтры" для отображения всех записей

Редактирование записей
Дважды кликните на запись в таблице для редактирования

Или выберите запись и нажмите "Редактировать запись"

Внесите изменения и нажмите "Сохранить изменения"

Удаление записей
Выберите запись в таблице

Нажмите кнопку "Удалить запись"

Подтвердите удаление в появившемся диалоговом окне

Просмотр статистики
В нижней части окна отображается статистика по отфильтрованным записям

Показывается количество записей, средняя, максимальная и минимальная температура

Отображается количество дней с осадками

Примеры использования
Пример 1: Добавление записей за неделю
Добавьте запись: 15.01.2024, температура -5°C, "Снежно, ветрено", осадки: да

Добавьте запись: 16.01.2024, температура -2°C, "Пасмурно", осадки: нет

Добавьте запись: 17.01.2024, температура -8°C, "Ясно, морозно", осадки: нет

Добавьте запись: 18.01.2024, температура +1°C, "Оттепель, гололед", осадки: да

В таблице отобразятся все 4 записи

Пример 2: Фильтрация по температуре
В поле "Температура от" введите: 0

Нажмите "Применить фильтр"

Отобразится только запись от 18.01.2024 с температурой +1°C

Статистика покажет: 1 запись, средняя температура +1.0°C

Пример 3: Фильтрация по дате и осадкам
В поле "Дата" введите: 17.01.2024

В поле "Осадки" выберите: "нет"

Нажмите "Применить фильтр"

Отобразится запись о ясной морозной погоде без осадков

Пример 4: Тестирование валидации
Дата: введите 31.13.2024 → ошибка "Дата должна быть в формате ДД.ММ.ГГГГ!"

Температура: введите "тепло" → ошибка "Температура должна быть числом!"

Описание: оставьте поле пустым → ошибка "Описание погоды не может быть пустым!"

Все поля: оставьте все поля пустыми → ошибка "Все поля должны быть заполнены!"

Пример 5: Использование всех фильтров одновременно
В фильтре "Дата" введите: 16.01.2024

В фильтре "Температура от" введите: -3

В фильтре "Осадки" выберите: "нет"

Нажмите "Применить фильтр"

Отобразится запись о пасмурной погоде без осадков с температурой -2°C
